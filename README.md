# Frog-simulation

# рЃљрЃЏрЃцрЃўрЃЉрЃўрЃћрЃЉрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃўрЃА рЃЏрЃљрЃарЃћрЃњрЃБрЃџрЃўрЃарЃћрЃЉрЃћрЃџрЃў рЃЊрЃљ рЃАрЃљрЃАрЃБрЃюрЃЌрЃЦрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃљрЃњрЃљрЃюрЃЏрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃџрЃЮ рЃЋрЃћрЃЉ-рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃАрЃўрЃЏрЃБрЃџрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃАрЃљрЃ«рЃљрЃЋрЃА рЃљрЃЏрЃцрЃўрЃЉрЃўрЃћрЃЉрЃўрЃА (рЃЉрЃљрЃДрЃљрЃДрЃўрЃА) рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃБрЃџ рЃЊрЃљ рЃАрЃљрЃАрЃБрЃюрЃЌрЃЦ рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃА рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ.

## ­Ъј» рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃў

рЃЊрЃћрЃЏрЃЮрЃюрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃфрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА:

- ­ЪљИ рЃЉрЃљрЃДрЃљрЃДрЃўрЃА рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљрЃќрЃћ.
- ­Ъње рЃљрЃЏрЃцрЃўрЃЉрЃўрЃћрЃЉрЃў рЃАрЃБрЃюрЃЌрЃЦрЃљрЃЋрЃћрЃю рЃарЃЮрЃњрЃЮрЃарЃф рЃърЃўрЃарЃўрЃЌ, рЃўрЃАрЃћ рЃЎрЃљрЃюрЃўрЃЌ.
- ­Ъїі рЃгрЃДрЃљрЃџрЃерЃў рЃЕрЃљрЃФрЃўрЃарЃЋрЃўрЃАрЃљрЃА рЃЉрЃљрЃДрЃљрЃДрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЎрЃљрЃюрЃўрЃЌ рЃАрЃБрЃюрЃЌрЃЦрЃЋрЃљрЃА.

## РџЎ№ИЈ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

- ­Ъћў **рЃгрЃДрЃљрЃџрЃерЃў рЃЕрЃљрЃФрЃўрЃарЃЋрЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃў** РђЊ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃњрЃљрЃЊрЃљрЃарЃЌрЃЮрЃА рЃЉрЃљрЃДрЃљрЃДрЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ: рЃгрЃДрЃџрЃўрЃА рЃЦрЃЋрЃћрЃе рЃљрЃю рЃњрЃљрЃарЃћрЃЌ.
- ­ЪїА№ИЈ **рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃЮрЃЉрЃљ** РђЊ рЃАрЃџрЃљрЃўрЃЊрЃћрЃарЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ рЃфрЃЋрЃџрЃўрЃљрЃю рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљрЃА 0┬░C-рЃЊрЃљрЃю 45┬░C-рЃЏрЃЊрЃћ.
- ­ЪЊЅ **рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃўрЃА рЃўрЃюрЃћрЃарЃфрЃўрЃљ** РђЊ рЃЉрЃљрЃДрЃљрЃДрЃўрЃА рЃАрЃ«рЃћрЃБрЃџрЃўрЃА рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃљ рЃюрЃћрЃџрЃљ рЃћрЃ«рЃЏрЃўрЃљрЃюрЃћрЃЉрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў (рЃЊрЃљрЃДрЃЮрЃЋрЃюрЃћрЃЉрЃўрЃЌ 5-10 рЃгрЃљрЃЏрЃў).


## ­ЪЊѓ рЃцрЃљрЃўрЃџрЃћрЃЉрЃў

- `index.html` РђЊ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЊрЃљ рЃћрЃарЃЌрЃљрЃЊрЃћрЃарЃЌрЃў рЃцрЃљрЃўрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃАрЃарЃБрЃџ рЃАрЃўрЃЏрЃБрЃџрЃљрЃфрЃўрЃљрЃА.

## ­Ъџђ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

1. рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ `index.html` рЃцрЃљрЃўрЃџрЃў.
2. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃЌ (Chrome, Firefox, Edge).
3. рЃЏрЃљрЃарЃЌрЃћрЃЌ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў рЃЊрЃљ рЃЊрЃљрЃљрЃЎрЃЋрЃўрЃарЃЊрЃўрЃЌ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА.
